• Iteration using for, while, and do-while loops </br>

These structures allow programs to make decisions and repeat tasks efficiently.

🧩 Functions</br>
I developed the ability to:</br>

• Use parameters and return values </br>
• Implement function overloading </br>
• Separate logic into reusable modules </br>
• Understand scope and lifetime of variables </br>
Functions improved the structure, readability, and maintainability of my programs.

📋 Arrays</br>
I gained experience working with:</br>
• One-dimensional arrays </br>
• Two-dimensional arrays </br>
• Traversing arrays using loops </br>
• Performing calculations on grouped data </br>

🔤 Strings and Arrays</br>
I learned how to:</br>
• Work with character arrays (C-style strings) </br>
• Use the C++ string class </br>
• Perform string manipulation (concatenation, comparison, searching) </br>

🧱 Structs and Arrays</br>
I developed knowledge in:</br>
